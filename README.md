# Rust Comprehensive

A hands-on learning repository for practicing and mastering the basics of Rust.

This project is designed for developers who want to strengthen their understanding of Rust through small, focused exercises. Instead of overwhelming yourself with large applications, this repository encourages learning by building simple programs that demonstrate core language concepts clearly.

## Why this repository exists

Rust is a powerful systems programming language with a strong emphasis on safety, performance, and clarity. This repository serves as a personal study space to:

- practice Rust syntax and fundamentals
- experiment with small examples
- build confidence through repetition and incremental learning
- document progress as new concepts are explored

## What you will find here

The repository currently contains beginner-friendly exercises under the Chapter 1 section, including examples that cover:

- functions and return values
- variables and block expressions
- arrays and indexing
- formatting and printing output

## Repository structure

- chapt-1/chapt-1-exercise-1 — a simple function-based arithmetic example
- chapt-1/chapt-1-exercise-2 — an example using block expressions and debug output
- chapt-1/chapt-1-exercise-3 — a small program demonstrating arrays and formatting

## Getting started

Make sure Rust is installed on your machine. If not, install it from the official Rust website:

https://www.rust-lang.org/tools/install

Once Rust is available, you can run any exercise directly with Cargo.

### Run an exercise

From the repository root, use one of the following commands:

```bash
cargo run --manifest-path chapt-1/chapt-1-exercise-1/Cargo.toml
cargo run --manifest-path chapt-1/chapt-1-exercise-2/Cargo.toml
cargo run --manifest-path chapt-1/chapt-1-exercise-3/Cargo.toml
```

## Learning goals

This repository is meant to help you:

- become comfortable with Rust syntax
- understand basic control flow and data structures
- practice writing and running small Rust programs
- develop a strong foundation before moving to more advanced topics

## Notes

Each exercise is intentionally small and focused so that the core ideas remain easy to understand. As your knowledge grows, this repository can be expanded with new chapters, exercises, and projects.
